An ecommerce store built with MERN stack.

This ecommerce store enable two main different flows or implementations:

1. Buyers browse the store categories, products and brands

2. Admins can edit, delete a category, product



Features of the App

  • Node provides the backend environment for this application
  • Express middleware is used to handle requests, routes
  • Mongoose schemas to model the application data
  • React for displaying UI components
  • Payment method from braintree(PayPal) solution to enable checkout and collecting payments
